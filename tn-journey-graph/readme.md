# TN Journey Graph

Author: Techn  
Version: 0.2.9  
Status: MVP

## Purpose

Front-end journey exploration for authorised users, powered by completed Independent Analytics sessions.

## Key Features

- Authorised front-end Journey Explorer drawer.
- Dynamic journey hop tabs and reusable histogram panels.
- Background session queue and aggregate journey graph tables.
- Protected WordPress REST endpoint.
- Settings, manual processing, and processing status.
- GitHub release updater support.

## Folder Structure

```text
functions/
scripts/
styles/
templates/
```

## Important Notes

This plugin reads existing Independent Analytics data. It does not add tracking, cookies, or a new identity system.
